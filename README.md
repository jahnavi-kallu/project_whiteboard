<h1>Real-Time Collaborative Whiteboard</h1>
<h2>Overview</h2>
A real-time collaborative whiteboard platform that enables multiple users to draw, edit, and interact simultaneously on a shared canvas. The application supports secure authentication, persistent storage, and low-latency communication for seamless collaboration.
<h2>Features</h2>
<ul>
<li> Real-time multi-user collaboration using Socket.IO</li>
<li>JWT-based user authentication</li>
<li>Create, save, and manage whiteboard sessions</li>
<li>Persistent storage using MongoDB</li>
<li>Responsive and intuitive user interface</li>
<li>Deployed on Vercel (frontend) and Render (backend)</li>
</ul>
<h2>Tech Stack</h2>
<h3>Frontend</h3>
HTML | CSS | JavaScript
<h3>Backend</h3>
Node.js | Express.js
<h3>Database</h3>
MongoDB
<h3>Authentication</h3>
JWT
<h3>Real-Time Communication</h3>
Socket.IO
